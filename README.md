## Overview
A production-ready dashboard template built with the new App Router, Tailwind CSS, and a collection of custom reusable components.

## Tech Stack
Next.js 14, TypeScript, Tailwind CSS, Lucide React

## Features
- Fully responsive Dashboard Layout (Sidebar, Header)
- Reusable UI elements (Button, Input, Table, Modal)
- Secure `/login` mock authentication portal
- Interactive `/dashboard/users` list with simulated filtering & pagination
- Strict environment configurations

## Setup
npm install
cp .env.example .env.local
npm run dev
